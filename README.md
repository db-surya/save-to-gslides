# Chrome Extension - Save to Google Slides

Chrome Extension that allows users to create presentations on Google Slides on the go. By simply selecting text or images from any webpage, you can create a new slide in Google Slides with just a few clicks. The selected text is automatically formatted as bullet points, and if the content exceeds the slide, it will automatically flow onto the next slide.
